
### Joomla! Programmers Documentation

The page introduces the Joomla! Programmers Documentation, a developer manual for building Joomla extensions such as components, modules, libraries, plugins, and templates. It explains that the documentation is meant for both Joomla core developers and third-party extension developers, notes that the manual is still incomplete, and invites users to contribute through GitHub edits or local pull requests.

https://manual.joomla.org/docs/
